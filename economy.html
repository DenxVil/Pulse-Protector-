ROI Calculation Methodology

// Lives saved calculation const traumaDeaths = 1500000; // Annual trauma deaths in India const prehospitalDeaths = traumaDeaths * 0.72; // 72% before hospital const preventableRate = 0.311; // 31.1% preventable const kitCoverage = 10000 / 1000000; // 10,000 kits for 1M population const livesSaved = prehospitalDeaths * preventableRate * kitCoverage; // Result: ~3,110 lives saved annually const economicValue = livesSaved * 140000; // ‚Çπ1.4L per life const investment = 10000 * 2500; // ‚Çπ2.5Cr investment const roi = economicValue / investment; // 186.6x return
