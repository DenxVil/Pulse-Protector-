PTT-Based Blood Pressure Estimation

Non-invasive BP measurement using Pulse Transit Time from ECG and PPG

How PTT Measures Blood Pressure

Pulse Transit Time (PTT) is the time it takes for a pulse wave to travel between two arterial sites. We measure PTT as the time delay between the R-peak of the ECG (cardiac electrical activity) and the arrival of the pulse wave at the finger (detected by PPG sensor). This delay is inversely related to blood pressure - higher BP = faster pulse wave = shorter PTT.

Implementation Steps:

  1. ECG Acquisition: Detect R-peaks using INA128 ECG front-end
  2. PPG Acquisition: Capture pulse wave arrival via MAX30102
  3. PTT Calculation: Measure time difference between R-peak and PPG foot
  4. Calibration: Use reference BP measurement to determine Œ± and Œ≤ coefficients
  5. BP Estimation: Apply coefficients to convert PTT to systolic/diastolic BP

Core Algorithm (C++ Pseudocode):

// Detect R-peak from ECG
if (ecgSample > rPeakThreshold) {
    rPeakTime = currentTime;
}

// Detect PPG foot (pulse arrival)
if (ppgSample < ppgFootThreshold) {
    ppgFootTime = currentTime;
}

// Calculate PTT
ptt = ppgFootTime - rPeakTime;

// Estimate BP using calibrated coefficients
systolicBP = alphaSys / ptt + betaSys;
